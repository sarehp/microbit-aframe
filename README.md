Controlling an A-Frame scene with a BBC micro:bit
=================================================

INSTALL
  1) install bitio
       https://github.com/whaleygeek/bitio
  2) install pyautogui
       https://pyautogui.readthedocs.io/en/latest/

RUN
  python microbit-pyautogui
